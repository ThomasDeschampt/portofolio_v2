Mes projets.

Mon premier portfolio en 3D

Dans ce premier essais de Portfolio, assez original, j'ai voulu mettre en oeuvre mes compétences de modélisation et mes compétences en Three JS ; en représentant ma chambre en 3D

Pour réaliser ce projet j'ai utilisé les langages HTML et CSS pour la conception et la mise en forme du site, JavaScript pour mettre en place certaines fonctionnalités (comme la traduction) et pour intégrer de la 3d sur le site grâce à THREE JS. J'ai aussi utilisé Blender pour modéliser tous les objets 3d présents sur le site.

CSS HTML Three JS


Simulateur de réactions chimiques

Dans le cadre de son travail, mon père était amené à mettre en place de réactions chimiques sans avoir de moyen de connaitre à l'avance comment allait réagir les produits et en quels quantités les produits allaient être formés. Ce programme permis ainsi de prevoir à l'avance quelles quantité de réactifs seront consommés et quels produits seront formés

Pour réaliser ce projet j'ai utilisé les langages HTML et CSS pour la conception et la mise en forme du site, Python pour mettre toutes les étapes et calculs des réactions et Flask pour relier le code HTML avec l'algorithme Python.

CSS HTML Python Flask


My Timer

Etant passionné à la fois par le sport et l'informatique, ce projet a été pour moi l'occasion de pouvoir lier ces deux domaines de manière complémentaire. Ce site web, regrouppe ainsi différent type de chronomètre (Tabata, ForTime, AMRAP,...) pour facilité tout types de séances avec un outil adapté à chaque type de pratique.

Pour réaliser ce projet j'ai utilisé les langages HTML et CSS pour la conception et la mise en forme du site, et JavaScript pour mettre en place les chronomètres et leur animations.
